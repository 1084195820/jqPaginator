#jqPaginator分页组件#

项目首页：[http://jqPaginator.keenwon.com/](http://jqPaginator.keenwon.com/)
问题反馈：[http://keenwon.com/1124.html](http://keenwon.com/1124.html)

源码基于GruntJS构建，确保安装Node的情况下，在文件目录下运行

    npm install
    
然后运行

    grunt server
    
修改相应代码会自动刷新  
打包编译请运行

    grunt build
    
开发文件位于src目录，编译后的最终文件位于dist目录
